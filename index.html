What is Quantimus?

Quantimus is a machine-learning based program used for histological analysis of skeletal muscle. Developed as a plugin for the image analysis tool flika, Quantimus is able to able measure cross-sectional areas, marker flouresence, and identify centrally-nucleated fibers in muscle sections.
